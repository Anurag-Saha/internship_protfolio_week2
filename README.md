# Week 2: Styled Portfolio Website - Documentation

## Project Overview
Added CSS styling to Week 1 HTML portfolio with responsive design, hover effects, and modern layout.

## CSS Concepts Used
1. **CSS Variables** - Created color scheme with :root
2. **Flexbox** - Used for header, navigation, and skills layout
3. **CSS Grid** - Can be added for more complex layouts
4. **Responsive Design** - Media queries for tablet and mobile
5. **Hover Effects** - Added to links, buttons, and sections
6. **Transitions & Animations** - Smooth hover effects

## Design Decisions
- **Color Scheme**: Blue primary, dark secondary, orange accent
- **Typography**: Poppins font from Google Fonts
- **Layout**: Mobile-first responsive approach
- **Effects**: Subtle shadows and smooth transitions

## Responsive Breakpoints
- Desktop: > 768px
- Tablet: 481px - 768px
- Mobile: < 480px
